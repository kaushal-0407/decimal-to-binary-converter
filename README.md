# decimal-to-binary-converter

A command-line Decimal to Binary Converter built using Java Programming. The program takes a decimal number as input from the user and converts it into binary using the division-by-2 method.
This project helped me strengthen my understanding of Java fundamentals such as loops, conditionals, strings, and StringBuilder.

## Features

- Converts decimal numbers into binary
- Accepts user input using Scanner
- Uses division-by-2 conversion logic
- Handles edge cases like 0 and 1
- Uses StringBuilder.reverse() for final output
- Simple command-line interface

## Concepts Used

- Java Basics
- Scanner Class
- Conditional Statements
- do-while Loop
- Modulus Operator (%)
- String Manipulation
- StringBuilder

## How It Works

- Enter a decimal number
- The program repeatedly divides the number by 2
- Stores the remainder after each division
- Reverses the collected remainders
- Displays the binary representation

## Requirements

- Java
- JDK 8 or above
